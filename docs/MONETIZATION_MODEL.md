## 1. EXECUTIVE SUMMARY

### 1.1 Model Bisnis

Tarombo Digital mengadopsi model **freemium dengan multiple revenue streams** yang menyeimbangkan:
- **Akses universal**: Core genealogi data gratis untuk preservasi budaya
- **Value-added services**: Fitur premium untuk pengguna yang membutuhkan lebih
- **B2B services**: Layanan untuk komunitas adat, peneliti, dan institusi

### 1.2 Value Proposition

| Stakeholder | Value Proposition |
|-------------|-------------------|
| **Individual Users** | Kenali silsilah, temukan kerabat, jaga tradisi |
| **Tetua Adat** | Kelola data komunitas, validasi adat digital |
| **Komunitas Batak** | Preservasi budaya, pendidikan generasi muda |
| **Peneliti** | Akses data genealogi terstruktur |
| **Pemerintah** | Dokumentasi warisan budaya digital |

## 3. PRICING STRATEGY

### 3.1 Pricing Principles

1. **Affordability**: Harga harus terjangkau untuk masyarakat Batak di semua level ekonomi
2. **Value-Based**: Harga berdasarkan nilai yang diberikan, bukan sekadar cost-plus
3. **Cultural Respect**: Tidak memonetisasi aspek sakral adat
4. **Transparency**: Tidak ada hidden fees

### 3.2 Geographic Pricing

| Region | Factor | Example (Premium) |
|--------|--------|-------------------|
| **Indonesia** | 1.0x | Rp 49.000 |
| **Malaysia/Singapore** | 1.5x | Rp 73.500 |
| **Netherlands** | 2.0x | Rp 98.000 |
| **USA** | 2.5x | Rp 122.500 |

### 3.3 Promotional Pricing

| Promotion | Discount | Conditions |
|-----------|----------|------------|
| **Early Adopter** | 50% | First 1000 users |
| **Student** | 30% | Valid student ID |
| **Senior (60+)** | 40% | Age verification |
| **Annual Commitment** | 20% | Yearly billing |
| **Referral** | 1 month free | Per successful referral |
