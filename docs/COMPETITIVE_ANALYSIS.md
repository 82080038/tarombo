## 2. APLIKASI GENEALOGY GLOBAL

### 2.1 Ancestry.com

| Atribut | Detail |
|---------|--------|
| **Founded** | 1983 (Lehi, Utah, USA) |
| **Users** | 30+ juta |
| **Database** | 30 miliar records |
| **DNA Database** | 22 juta |
| **Revenue** | $1.4B (2023) |

**Features:**
- Family tree builder dengan hints AI
- DNA ethnicity estimates (3,600+ regions)
- Historical records search
- Collaboration tools
- Mobile app (iOS/Android)

**Pricing:**
| Paket | Harga | Fitur |
|-------|-------|-------|
| **U.S. Discovery** | $24.99/bulan | US records only |
| **World Explorer** | $39.99/bulan | Global records |
| **All Access** | $49.99/bulan | + Newspapers.com, Fold3 |
| **DNA Kit** | $99.00 (sekali) | + ethnicity estimate |

**Strengths:**
- Database terbesar
- UX/UI polished
- Gamification dengan hints
- Strong brand recognition

**Weaknesses (Relevan untuk Tarombo Digital):**
- ❌ Tidak mendukung sistem kekerabatan patrilineal kaku
- ❌ Tidak ada konsep marga/klan
- ❌ Tidak ada partuturan/address system
- ❌ Tidak ada validasi adat
- ❌ Western-centric, tidak mengerti konteks budaya Batak
